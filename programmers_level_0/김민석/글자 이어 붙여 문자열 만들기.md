## 글자 이어 붙여 문자열 만들기

```java
class Solution {
    public String solution(String my_string, int[] index_list) {
        String answer = "";
        String[] str = my_string.split("");
        for (int i = 0; i < index_list.length; i++) {
            answer += str[index_list[i]];
        }
        return answer;
    }
}
```
