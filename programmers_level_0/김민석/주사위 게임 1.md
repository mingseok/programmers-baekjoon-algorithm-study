## 주사위 게임 1


```java
import java.util.*;

class Solution {
    public int solution(int a, int b) {
        int answer = 0;
        
        if(a % 2 == 1 && b % 2 == 1) {
            answer = (a * a) + (b * b);
        } else if((a % 2 == 1 || b % 2 == 1)) {
            answer = 2 * (a + b);
        } else if(a % 2 == 0 && b % 2 == 0) {
            answer = Math.abs(a - b);
        }
        
        
        return answer;
    }
}
```
