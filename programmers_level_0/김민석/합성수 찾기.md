## 합성수 찾기

```java
class Solution {
    public int solution(int n) {
        int answer = 0;
        
        for (int i=1; i<=n; i++) {
            int a = 0;
            for (int j = 1; j<=n; j++) {
                if (i%j == 0) {
                    a++;
                }
            }
            if (a >= 3) {
                answer++;
            }
        }
        return answer;
    }
}
```
